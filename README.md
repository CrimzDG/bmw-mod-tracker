# 🚗 BMW Mod Tracker

A full-stack app to track modifications on your BMW — with user auth, car management, and a Kanban board for mods.

## Tech Stack

| Layer | Tech |
|---|---|
| Backend | Spring Boot 3, Java 21, Spring Security, JWT |
| Database | PostgreSQL (Supabase) |
| Frontend | React + Vite |
| Hosting | Railway (backend), Vercel (frontend) |

## Project Structure

```
bmw-mod-tracker/
├── backend/     # Spring Boot API
└── frontend/    # React app
```
